# Himasha Indiwari - Portfolio

A modern, professional Software Engineering portfolio built with React, Vite, Tailwind CSS, Framer Motion, and React Icons.

## Tech Stack

- **React 19** + **Vite 8**
- **Tailwind CSS 4**
- **Framer Motion** — animations & transitions
- **React Icons** — icon library

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Customization

Update your personal details, projects, and social links in:

```
src/data/portfolioData.js
```

Place your CV file at `public/cv.pdf` for the Download CV button.

## Features

- Dark theme with cyan/blue accents
- Fully responsive (mobile, tablet, desktop)
- Smooth scroll navigation with active section highlighting
- Section reveal animations
- Typing effect in hero section
- Animated skill & project cards
- Project filtering with animations
- Scroll progress indicator
- Back to top button
- Loading screen animation
- Glassmorphism UI effects
- Contact form (opens mail client)

## Project Structure
